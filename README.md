# TestFlix2022
APIs, Collections, k6 Scripts used in TestFlix2022


**k6:**

Converting Postman collections to k6 scripts

**Prerequisites:**
* Download and Install k6 open source
* Prebuilt Postman collections
* Install Node.js
* Install postman-to-k6 converter
